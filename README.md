# Picture-To-ASCII
Turns a picture into ascii image


**How to use?**
1. Clone the repo
2. Run ``npm install``
3. Add a picture to the folder
4. Run ``npm start``
5. Enter ``./<file name with extension>``
6. Then there will be a ``output.txt`` file, open it and zoom out until you can see the image
